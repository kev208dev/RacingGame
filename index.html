        조작법 (Controls)

        W / S가속 / 브레이크 · 후진
        A / D좌 / 우 조향
        SPACE핸드브레이크 — 한 번 누르면 일반 드리프트, 두 번 연속(0.3초 이내)이면 U턴급 더블 드리프트
        SHIFT부스트 발동 / DRX 직선 구간에서는 더블클릭으로 슈퍼부스트
        Q / E기어 업 / 다운 (8단 수동 모드 자동 전환)
        T자동 ↔ 수동 변속 토글
        R차량 리셋 (스타트 라인으로)
        C카메라 시점 전환 (체이스 / 보넷 / 탑뷰)
        ESC메뉴로 나가기

        기술 시연

        SPACE 기본 드리프트

        코너 진입에서 한 번 살짝 누르면 자연스러운 슬립. 부스트 미터 충전용.

        SPACE×2 더블 드리프트

        U턴 / 헤어핀에서 사용. 짧게 두 번 톡톡 누르면 약 90° 임펄스로 차체가 휙 돌아감.

        SHIFT 부스트

        미터가 30 이상일 때 한 번 누르면 1.5초간 최고속·가속 폭증.

        SHIFT×2 슈퍼부스트 (DRX)

        직선 구간에서 DRX 게이지가 살아있을 때 SHIFT를 두 번 톡톡. 2.6초간 압도적 가속.

        기능 안내

        • 드리프트 — 핸드브레이크 + 코너에서 슬립하면 BOOST 미터가 차오름
        • 더블 드리프트 — 스페이스를 짧은 간격으로 두 번 누르면 약 90° 임펄스가 발동해 U턴 / 헤어핀에서 한방 정렬
        • 부스트 — 30 이상 차면 SHIFT 한 번에 1.5초간 최고속+가속 ↑
        • 기어 — 자동(AUTO): 알아서 변속. 수동(MANUAL): 레브 리미터 + 다운시프트 보호
        • 벽 충돌 — 4개 바퀴 모두 트랙 안쪽으로 유지. 빠른 충돌은 화면 흔들림 + 스파크
